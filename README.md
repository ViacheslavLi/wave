# Requirements:

Python 3.6 or later with the following packages installed:  

- ```numpy```
- ```scipy```
- ```pytorch``` >= 0.4.0 
- ```plotly``` (optional)

# Running WAVE:
1. Make sure that all of the requirements are met prior to running WAVE:
```!pip3 install numpy scipy pytorch plotly --update```

2. Run ```wave.py```
